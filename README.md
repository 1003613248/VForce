# Bin-Picking
Perception system using RealSense Camera for bin-picking

## Dependence
1. librealsense(<2.0)
2. PCL(Point Cloud Library)(>=1.7)
3. OpenCV(>=3.0.0)
4. PythonLibs(>=3.5)
6. Glog
7. Gflag

## How to build
```bash
cd PROJECT_DIR
mkdir build
cd build
cmake ..
make
```
